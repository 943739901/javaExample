#### equls 和 == 的区别
基本数据类型 == 比较的是实际值 
复合数据类型 == 比较的是他们在内存中的存放地址 
复合数据类型中 equal (string lngeter date)等重写了 equal，比较的是内容 没有重写 equal的，比较的还是内存地址 

StringBuffer 和 StringBuilder 特殊，==和 equal 都是比较地址
